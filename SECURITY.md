## Reporting a Vulnerability

If you discover a potential security issue, please follow these steps:

1. **Open an issue** on this repository with a clear description of the vulnerability.  
2. **Sensitive information:** Do not post sensitive data publicly; request private communication if needed.  
3. **Third-party dependencies:** If the issue involves a library, framework, or other external dependency, also report it to the respective maintainers.  
4. **Response:** The maintainers will review and address the issue as appropriate.
